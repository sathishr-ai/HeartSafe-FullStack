TROUBLESHOOTING:
----------------

Q: Backend won't start
A: Make sure Node.js is installed, run "npm install" first

Q: Frontend can't reach backend
A: - Ensure backend is running on port 5000
   - Check browser console for CORS errors
   - Frontend will fallback to local calculation if backend unavailable

Q: Login fails
A: - Use exact credentials listed above
   - Username is case-sensitive
   - Check browser console for error messages

Q: Predictions not working
A: - Make sure backend server is running
   - Check that all form fields are filled
   - Look at browser console for API errors

Q: CSV upload fails
A: - Make sure CSV has correct columns: age, gender, cholesterol, hdl, systolic, diastolic, smoking, diabetes, family
   - Use the downloaded template as reference
   - Check file size is reasonable
